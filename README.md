# ClassOneSystem Replica

Local React + Vite replica of [classonesystems.in](https://classonesystems.in/) with placeholder imagery.

## Run locally

```bash
npm install
npm run dev
```

The app is configured to run on `http://localhost:5173`.

## Notes

- All images are replaced with structured placeholders.
- The homepage, major navigation pages, and common product/software routes are included.
- Internal links stay in the local app so the replica can be explored without leaving the workspace.
